Script en Python que permite descargar audio de YouTube en la mejor calidad disponible, incluyendo opciones sin pérdida cuando existen. Cuenta con una interfaz gráfica básica para facilitar su uso sin utilizar la terminal.

Características

Descarga audio en máxima calidad.

Soporte para formatos sin pérdida (si el video los ofrece).

Interfaz gráfica simple e intuitiva.

Selección de carpeta de destino.

Funcionamiento rápido y directo.

Requisitos

Python 3.x

yt-dlp (o la librería que utilices)

tkinter para la interfaz (incluido en la mayoría de instalaciones de Python)

Uso

Ejecuta el archivo script.py.

Ingresa la URL del video de YouTube.

Selecciona la carpeta donde guardar el audio.

Haz clic en “Descargar”.

Nota legal

Este proyecto es solo con fines educativos. Respeta los términos de servicio de YouTube y las leyes de derechos de autor de tu país antes de descargar contenido.
